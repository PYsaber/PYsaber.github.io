Hexo-Algolia配置错误

发表于 2017-06-07 | 分类于 杂谈 | | 阅读次数

今天在给本站完善功能时，在添加搜索功能时遇到了一些小问题，在这里记录一下踩的坑

  • Hexo版本:3.2.2
  • Next版本:5.1.0

今天根据Next的官方文档给本站集成搜索功能时，看到官方提供了好几个搜索插件，选了最新的Algolia插件，
然而在配置的时候出现了问题。这是因为3.2.2版本的Hexo在package.json文件的配置中存在错误。到Hexo的根目录，在其中找到package.json文件，修改其中的hexo-algolia属性值为^0.2.0。然后重新安装hexo-algolia即可。还有一个点要注意，Next配置文件中algolia的apiKey属性指的是algolia服务提供的Search-Only API Key
